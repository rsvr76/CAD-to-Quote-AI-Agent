# Step 8 — ML Prediction

**Phase:** 4 — Prediction  
**Input:** 17-feature vector (from Step 7)  
**Output:** Predicted machining time (minutes)

---

## Goal

Run the trained Random Forest model on the assembled feature vector to predict **machining time** for the part.

Correctness requirement:

> The feature vector schema used here is identical to the schema used during model training to ensure consistent inference behavior.

---

## Key Design Decision

The model predicts **machining_time** (minutes), NOT machining cost (₹).

Cost is derived later in Step 10:

```
machining_cost = machining_time × machine_rate
```

This separation ensures:
- The model learns physical relationships (time), not financial ones (cost)
- Machine rates can change without retraining the model
- Different machine rates can be applied per selected machine/process in the cost engine

---

## Implementation

```python
import joblib
import pandas as pd

# Load once at service startup (avoids repeated disk I/O)
MODEL_PATH = "models/machining_model.joblib"
model = joblib.load(MODEL_PATH)

# Must match Step 7 + training order exactly
FEATURE_ORDER = [
    "surface_area_cm2",
    "volume_cm3",
    "aspect_ratio",
    "sv_ratio",
    "slenderness",
    "pocket_density_proxy",
    "num_holes",
    "num_pockets",
    "max_depth_mm",
    "material",
    "tolerance_class",
    "is_turning",
    "is_milling",
    "is_drilling",
    "is_grinding",
    "complexity_score",
    "drilling_work",
]

def predict_machining_time(features: dict, *, min_minutes: float = 0.5) -> float:
    """Predict machining time using the trained Random Forest model."""
    # Build a 2D input with a guaranteed column order.
    # (Never rely on dict ordering for ML inference.)
    row = [features[name] for name in FEATURE_ORDER]
    feature_df = pd.DataFrame([row], columns=FEATURE_ORDER)
    
    # Predict
    machining_time = float(model.predict(feature_df)[0])

    # Sanity floor: protects downstream cost math from negative/zero outputs.
    machining_time = max(min_minutes, machining_time)
    return round(machining_time, 2)
```
