# Step 6 — User Stock Confirmation

**Phase:** 3 — Cost Engine  
**Input:** Estimated stock weight (from Step 5)  
**Output:** Final confirmed stock weight

---

## Goal

Display the estimated stock weight to the engineer and capture the **final confirmed** stock weight.

If the engineer already adjusted the value earlier (for example via the Step 5 dashboard), this step is **prefilled** with that value for a final check.

Architecture note:

> The final stock weight is treated as a deterministic input to the cost engine and is **not predicted by the ML model**.

---

## Why Confirmation Is Needed

The stock estimation in Step 5 uses bounding box geometry with margins. Real stock may differ because:

- Standard bar/plate sizes may not match the calculated dimensions
- Off-cuts or pre-existing stock may be used
- Material supplier constraints may dictate different stock sizes

This step is the **single final checkpoint** before material cost is calculated.

---

## UI Interaction

```
System: Estimated stock
    Shape: rectangular block (Turning OFF)
    Dimensions: 130 × 50 × 40 mm
    Weight: 1.6 kg (Steel)
        Does this match your stock? [Accept / Override]

Engineer: Override → 1.8 kg
```

If overridden (in Step 5 or Step 6), the UI records it as an **Engineer override** (optionally with a short note).

If the override differs significantly from the estimate (example threshold: > 30%), show a gentle warning:

> Override differs significantly from the estimate. Please re-check stock dimensions and units.

## Implementation

```python
def confirm_stock_weight(
    estimated_weight_kg: float,
    user_override_kg: float | None = None,
    *,
    min_kg: float = 0.01,
    max_kg: float = 100.0,
    warn_ratio_threshold: float = 0.30,
):
    """Return final stock weight with basic validation + audit info."""
    def _validate(value_kg: float, label: str) -> None:
        if value_kg is None:
            return
        if value_kg <= 0:
            raise ValueError(f"{label} must be positive")
        if not (min_kg <= value_kg <= max_kg):
            raise ValueError(f"{label} must be between {min_kg} and {max_kg} kg")

    _validate(estimated_weight_kg, "estimated_weight_kg")
    _validate(user_override_kg, "user_override_kg")

    final_weight_kg = user_override_kg if user_override_kg is not None else estimated_weight_kg
    override_applied = user_override_kg is not None

    warning = None
    if override_applied and estimated_weight_kg > 0:
        ratio = abs(final_weight_kg - estimated_weight_kg) / estimated_weight_kg
        if ratio > warn_ratio_threshold:
            warning = "Override differs significantly from estimate. Re-check stock dimensions/units."

    stock_info = {
        "estimated_weight_kg": estimated_weight_kg,
        "final_weight_kg": final_weight_kg,
        "override": override_applied,
        "warning": warning,
    }
    return final_weight_kg, stock_info
```
